# knowing。

A daily knowledge digest — beautifully designed, AI-generated, and endlessly curious.

🌐 **[curiositybox.vercel.app](https://curiositybox.vercel.app)**

---

## What it is

knowing。generates a fresh batch of surprising, well-researched facts and ideas across science, history, philosophy, culture, language, food, art, and more. Each card includes a key insight, source citation, and a shareable image you can export in multiple visual styles.

---

## Features

**Themes**
- 4 hand-crafted themes: Linen, Amber, Olive, Ocean
- Ocean is a full dark mode theme
- Auto-switches to Ocean when system is in dark mode

**Content**
- AI-generated knowledge cards via Anthropic Claude API
- 7 mood filters: Surprise, Dark, Funny, Contemplative, Beautiful, Unsettling, Any
- English / 中文 toggle per card
- Category filtering and keyword search
- Star and archive your favourites
- Balanced topic sampling across 10 categories to avoid repetition

**Save as Image**
- 8 visual styles: Light, Ink, Parchment, Gradient, Editorial, Category, Neon, Minimal
- 3 aspect ratios: square, portrait, landscape
- Justified typography, style-specific fonts and layouts

**Export**
- Export archive as a designed PDF — cover page with stats, one card per page
- JSON backup export and import (merges without duplicates)

**Sync**
- Supabase auth — sign in to sync starred cards across devices

**Extras**
- 🐦 Flappy Bird easter egg
- Knowledge Diet sidebar with curated reading links by category
- Random wonder popup
- Fun bits section: Pointless Wisdom, You Didn't Need to Know, Bad Idea of the Day
