# Curiosity Box

A daily knowledge digest — beautifully designed, AI-generated, and endlessly curious.

🌐 **[curiositybox.vercel.app](https://curiositybox.vercel.app)**

---

## What it is

Curiosity Box generates a fresh batch of surprising, well-researched facts and ideas across science, history, philosophy, design, and more. Each card includes a key insight, source, and a shareable image you can export in multiple visual styles.

---

## Features

**Themes**
- 5 hand-crafted themes: Linen, Amber, Olive, Ocean, Default
- Auto-switches to Ocean dark mode based on system preference

**Content**
- AI-generated knowledge cards via Anthropic API
- English / 中文 toggle per card
- Category filtering and keyword search
- Star and archive your favourites

**Save as Image**
- 8 visual styles: Light, Ink, Parchment, Gradient, Editorial, Category, Neon, Minimal
- 3 aspect ratios: square, portrait, landscape
- Justified typography, style-specific fonts and layouts

**Sync**
- Supabase auth — sign in to sync starred cards across devices

**Extras**
- 🐦 Flappy Bird easter egg
- Knowledge Diet sidebar with curated reading links
- Random wonder popup
